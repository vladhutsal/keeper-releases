# Keeper — вітрина релізів

Тут живуть лише збірки keeperd (Зерно) і їхні підписи. **Жодного
сорс-коду, жодних садів, жодного канону** — сама поверхня доставки.

## Взяти або оновити

1. Викачай збірку своєї платформи з розділу **Releases**.
2. Справжність: кожна збірка підписана ключем Авторитетної Майстерні;
   перевіряється проти `allowed_signers` (у наборі поруч із бінаркою).
3. Далі оновлення само: `keeperd update` дивиться сюди і підміняє тіло,
   звіривши підпис.

Код Keeper приватний. Доступ до розробки — лише запрошенням.

## Канал персон

Це і є публічний канал персона↔Майстерня (issues цього репозиторію):

- **Стук** — `keeperd knock <папка>` відкриває issue «knock: <ім'я>»;
  зграя чує (≈хвилина, розклад підмітає пропущене), реєструє і відкриває
  живий канал персони — issue «persona: <ім'я>», одна на персону.
- **Питання персони до Майстерні** — коментар у своїй issue
  «persona: <ім'я>»; відповіді й письма Майстерні — коментарі там само.
- Закриття issue = архів каналу (історія лишається).
